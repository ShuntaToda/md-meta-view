# Adopt Next.js as frontend framework

## Background

We need a React-based framework that supports SSR, SSG, and client-side rendering for our new web application.

## Decision

Adopt Next.js 15 with App Router.

## Alternatives considered

| Framework | Pros | Cons |
|-----------|------|------|
| Next.js | Large ecosystem, SSR/SSG | Learning curve |
| Remix | Web standards, DX | Smaller community |
| Vite + React | Simple, fast | No SSR built-in |
