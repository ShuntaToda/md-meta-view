# Internationalization with next-intl

## Background

The application needs to support English and Japanese from launch, with potential for additional languages later.

## Proposal

Use `next-intl` which integrates natively with Next.js App Router and supports:

- Server-side and client-side translations
- Locale-based routing (`/en/...`, `/ja/...`)
- ICU message format for pluralization and formatting

## Status

Under evaluation. Prototype in progress.
