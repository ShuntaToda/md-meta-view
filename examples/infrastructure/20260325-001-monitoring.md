# Monitoring and alerting with Datadog

## Background

We need observability across the full stack: application performance, error tracking, logs, and infrastructure metrics.

## Proposal

Use Datadog for unified monitoring:

- **APM**: Distributed tracing across API and database
- **Logs**: Centralized log aggregation with search
- **Alerts**: PagerDuty integration for on-call notifications
- **Dashboards**: Custom dashboards for key business metrics

## Alternatives

- CloudWatch: Free with AWS, but limited APM capabilities
- Grafana + Prometheus: Self-hosted, more operational overhead

## Status

Trial period started. Evaluating cost and integration effort.
